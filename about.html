알파버젼 개발 일지 (2020.06.09, ver.0.53a)

아래 진척도에서 ✓항목은 구현한 것,  -항목은 구현 예정입니다.

그 밖의 요청사항 있으시면 정재환에게 연락주세요.


개발 진척도

  • 1. 열전물성 입력
    • ✓ 물성테이블에 직접 입력: 줄 추가, 줄 선택, 줄 삭제.
    • ✓ 물성테이블에 붙여넣기: 테이블 클릭 후 Ctrl+V로 붙여넣을 수 있음.
  • 2. 열전물성 곡선공식 도출
    • ✓ 다항식 regression 가능: 1차~5차 다항식. Powell method로 최적화.
    • ✓ 다항식 interpolation 가능: Chebyshev nodes 이용.
    • ✓ Piecewise linear interpolation 가능. (2020.06.03)
    • ✓ 결과를 clipboard로 복사.
    • - 기타 regression model: 요청이 있으면 추가할 예정.
  • 3. 시뮬레이션 파라매터 입력
    • ✓ Leg spec 입력 가능: 길이, 면적, leg 갯수.
    • ✓ 소자구동 조건 입력 가능: 고온부/저온부 온도, 전류(ref. current) 영역.
    • ✓ Solver 선택 가능: Picard iteration 또는 Powell Method.
    • ✓ Solver 조건 선택 가능: 온도해 Chebyshev node 갯수, solver의 최대 반복 횟수, solver의 목표에러(tolerance), 전류 영역의 Chebyshev node 갯수.
  • 4. 열전발전 시뮬레이션 결과도출
    • ✓ 전류[A]-출력[mW] 커브 출력.
    • ✓ 전류[A]-효율[%] 커브 출력.
    • ✓ 시뮬레이션을 시작하기 전에, 이상한 물성에 대해서는 경고:
      • ✓ 측정 온도범위를 벗어나는 온도.
      • - 저온이 고온보다 높을 경우 (열냉각기 테스트를 위해 경고 보류).
    • ✓ 시뮬레이션의 결과의 오차를 예측해 출력.
    • ✓ 최대출력/최대효율 및 해당전류 값 출력.
    • ✓ 결과를 clipboard로 복사.
  • 5. About page
    • ✓ Members 사진/소개: 요청에 따라 지속적으로 업데이트 예정. CV 업로드나 개인 홈페이지 링크도 가능합니다.
    • - Web simulator의 계산방법론 소개.
    • - Contact 정보.
  • 6. 기타
    • ✓ Google Analytics 웹사이트 통계 기능 추가.
